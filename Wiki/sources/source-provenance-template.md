# Source Provenance Template

Use this block when creating or upgrading a source-derived wiki page.

```md
## Provenance
- Theme: `...`
- Registry: [[clipping-registry]]
- Inventory: [[clipping-inventory]]
- Raw-source verification: [[../workflows/raw-source-verification|Raw Source Verification]]

## Sources Included
- `Exact Raw Filename.md`
- `Another Exact Raw Filename.md`
```

## Guidance
- Keep exact filenames unchanged.
- Add only the files that materially influenced the page.
- If the page cites specific numbers, benchmarks, commands, or protocol details, confirm them from the raw clipping before finalizing the page.
