# Sources Hub

Use this directory for short project-specific summaries of external references in `Clippings/`.

Do not duplicate the full source text. Instead record:
- why the source matters
- the few takeaways relevant to this project
- how it changes routing, workflows, security, or implementation choices
- links to related wiki pages

Suggested filename style:
- `mcp-overview.md`
- `routing-papers.md`
- `rlhf-notes.md`
- `envoy-gateway-notes.md`

Prioritize summarizing only the sources that are actively shaping decisions.
